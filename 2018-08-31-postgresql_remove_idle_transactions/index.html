Remove idle transactions

l-lin included in post
 2018-08-31  21 words   One minute 
1
2
3
4

SELECT pg_terminate_backend(pid) 
FROM pg_stat_activity 
WHERE state LIKE 'idle %' 
AND state_change < now() - interval '1 hour';
